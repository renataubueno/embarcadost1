DESCRIPTION

Network interface driver for a simple interface (queue based) attached to an Hermes NoC. This driver works with 16-bit flits. Basic media access functions (_ni_status(), _ni_read() and _ni_write()) defined in the processor architecture memory map are used in this driver.

Packet format is as follows:

 2 bytes   2 bytes         ....
--------------------------------------
|tgt_cpu  |payload  |  ... data ...  |
--------------------------------------
